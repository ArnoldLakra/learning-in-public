# 🛠️ Git Learning Log - Day 1
Learned core version control concepts today from Apna College. 

### Commands Practiced:
- git init : Initialized a new local Git repository.
- git status : Checked the state of the working directory and staging area.
- git add <filename> : Staged specific files for the next commit.
- git add . : Staged all modified and new files at once.
- git commit -m "message" : Saved a snapshot of the staged changes with a descriptive comment.
- git push origin main : Uploaded local commits to the remote GitHub repository.
